# CrealityPrintPortable
Executable for running CrealityPrint in portable mode
Profile settings will be saved in `profile` folder

## Installation
- Download and unzip CrealityPrint: [https://github.com/CrealityOfficial/CrealityPrint/releases](https://github.com/CrealityOfficial/CrealityPrint/releases)
- Copy `CrealityPrintPortableStarter.exe` inside your CrealityPrint folder (next to the original `Creality-print.exe`)
- Run `CrealityPrintPortableStarter.exe`
